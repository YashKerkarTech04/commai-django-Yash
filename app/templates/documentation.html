Text Interface Documentation

The Text Interaction feature in CommAI allows users to engage in conversations with an AI model, facilitating human communication analysis and improvement.

1. Accessing the Text Interaction

Navigate to the Text Interaction option in the services section of the navigation bar on the homepage.

2. Chat Interface & Functionality

The interface consists of a chat screen and a text area where users can input messages. A send icon is located to the right of the textarea. Clicking this icon sends the user's message to the chat screen and interacts with the Gemini API for responses. The conversation continues as long as the user wishes, allowing AI-powered communication.

3. Submitting Conversations for Analysis

At the bottom right, there is a Submit Conversation button. Clicking this button processes the entire conversation for detailed analysis.

4. Conversation Analysis Features

Once submitted, the system evaluates the conversation based on seven key communication parameters:

  • Clarity Score
  • Conciseness Score
  • Tone & Sentiment Score
  • Engagement Score
  • Vocabulary Usage Score
  • Politeness Score
  • Grammar & Spelling Score

Each score is accompanied by improvement suggestions, making it easier for users to refine their communication skills.

5. Error Detection

The system highlights errors found in the user's conversation, providing correction suggestions in an interactive manner.

6. Facial Expression Analysis

Using CNN models built with TensorFlow and PyTorch, the system detects real-time facial expressions throughout the conversation.

7. AI-Powered Conversation Summary

Leveraging pretrained models like BERT and T5 Tokenizer, the system provides a concise summary of the entire conversation.

8. Conversation Level Classification

The user's communication skills are categorized as Poor, Intermediate, or Expert based on the seven key parameters.

9. Learning Resources

Depending on the user's level, a recommended improvement video is displayed along with AI-selected courses. The system employs a TF-IDF vectorizer and cosine similarity model to match user conversations with relevant learning materials.

10. Final Actions

  • Back to Home: Redirects the user to the CommAI homepage.
  • Download PDF: Generates and saves a communication progress report as communication_progress_report.pdf.

Daily Tasks Documentation

The Daily Tasks feature in CommAI helps users enhance their communication skills through structured daily exercises and AI-powered feedback.

1. Accessing Daily Tasks

Navigate to the Daily Tasks option in the services section of the navigation bar on the homepage.

2. Task Interface & Functionality

The interface displays a list of daily exercises that users can complete. Each task consists of either a text-based writing prompt or a speech-based challenge. Users can type their responses in the text area or use the microphone to submit voice responses, depending on the task type.

3. Submitting Tasks for Analysis

Once a task is completed, users can click the Submit Task button to process their response for detailed evaluation.

4. Task Analysis Features

After submission, the system evaluates the response based on seven key communication parameters:

  • Clarity Score
  • Conciseness Score
  • Tone & Sentiment Score
  • Engagement Score
  • Vocabulary Usage Score
  • Politeness Score
  • Grammar & Spelling Score (for text) / Grammar & Pronunciation Score (for speech)

Each score is accompanied by improvement suggestions to help users refine their communication skills.

5. Error Detection

The system highlights mistakes in the user's response, providing corrections for grammar, spelling, pronunciation, and structure.

6. Facial Expression Analysis (for Speech Tasks)

For speech-based tasks, the system detects real-time facial expressions using CNN models built with TensorFlow and PyTorch, ensuring expressive and engaging communication.

7. AI-Powered Task Summary

Utilizing pretrained models like BERT and T5 Tokenizer, the system generates a concise summary of the user's response.

8. Performance Level Classification

Users are categorized into Poor, Intermediate, or Expert levels based on their performance in the completed task.

9. Learning Resources

Based on the user's performance, a recommended improvement video is displayed along with AI-selected courses. The system employs a TF-IDF vectorizer and cosine similarity model to match the user's response with relevant learning materials.

10. Final Actions

  • Back to Home: Redirects the user to the CommAI homepage.
  • Download PDF: Generates and saves a progress report as daily_task_report.pdf.
